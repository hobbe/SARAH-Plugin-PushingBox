Le module pushingbox permet d'utiliser le service PushingBox pour envoyer des notifications sur le Cloud.

PushingBox est un service de notification en ligne. Il permet d'envoyer des notifications sur des services tels que :

  • Email
  • Twitter
  • Karotz
  • Android avec Notifry ou Notify My Android
  • iPhone avec Prowl ou Push 4.0
  • Windows Phone avec Toasty
  • URL personnalisées...

Le paramètre deviceid permet de véhiculer le scénario PushingBox à exécuter.

 

Pour configurer le module vous devez :

  • Créer un scénario et copier le DeviceID correspondant
  • Utiliser la variable $message$ dans le message de la notification. Cette variable sera remplacée par le texte (tts) généré par SARAH
  • Renseigner dans SARAH l'identifiant du DeviceID

 

Il est possible de masquer la vocalisation en ajoutant le paramètre quiet dans la définition de la règle.

options.quiet = "true";
